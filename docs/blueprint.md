# **App Name**: CredChain

## Core Features:

- Wallet Connection: Allow Issuers, Candidates, and Verifiers to connect their wallets.
- Credential Creation and Signing: Enable issuers to create credential data and sign it off-chain using EIP-712 signatures.
- Credential Hash Registration: Allow issuers to register the credential hash on the blockchain.
- Credential Viewing: Enable candidates to view credentials issued to them.
- Credential Sharing: Allow candidates to share their credential data or a verification link.
- Credential Verification: Allow verifiers to input credential data and verify its signature, issuer, and on-chain registration using a smart contract tool.
- Verification Result Display: Clearly display the verification result (Verified, Not Issued, Invalid Signature).

## Style Guidelines:

- Primary color: Red (#FF0000), conveying urgency and importance.
- Secondary color: Black (#000000), providing a stark contrast and a sense of sophistication.
- Accent color: Purple (#800080) adds a touch of luxury and mystery, used for interactive elements and highlights.
- Font: 'Inter', a sans-serif font known for its clear legibility and modern appearance, used for all text elements.
- Use simple, clear icons to represent actions and status, enhancing usability.
- Maintain a minimal, uncluttered layout with clear sections for each user role and action, ensuring ease of use.
- Use subtle animations to provide feedback on interactions (e.g., wallet connection, verification process) without being distracting.
